# BIST100 Assignment 1 - Strategy Backtesting and Optimization

## 1. Purpose
This project implements and compares two rule-based trading strategies on BIST100 using Python and Backtrader:

- Trend Following
- Mean Reversion (Contrarian)

The work follows the assignment requirements end-to-end:

- Data download from yfinance
- Daily return computation
- Backtrader strategy implementation (two separate classes)
- Parameter grid search for n=1..10 and m=1..10
- Performance evaluation (total return, annualized return, volatility, Sharpe, max drawdown)
- Best-parameter selection and strategy comparison

## 2. Final Answer to Assignment Question
For the tested sample period and setup, the optimized Trend Following strategy has higher Sharpe ratio than optimized Mean Reversion.

Result: It is better to be a trend follower than a contrarian in this specific backtest sample.

## 6. Important Data Note
The assignment ticker is `^XU100`. In yfinance, this symbol can be sparse/incomplete for longer history windows. The script therefore attempts `^XU100` first and uses `XU100.IS` if needed to ensure sufficient daily history for robust optimization.

The actual ticker used in the latest run is recorded in `results/data_diagnostics.csv`.
